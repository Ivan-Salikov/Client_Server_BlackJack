Требования:
python: не ниже версии 3.12.5

Для запуска нужно распаковать архив, затем через консоль зайти в папку проекта
Командой python -m venv .venv создаем виртуальную среду для проекта
Запускаем её командой .venv\Scripts\activate
Затем с помощью команды pip install requirements.txt устанавливаем все необходимые библиотеки
Для запуска программы через консоль заходим в папку blackjack и прописываем команду py manage.py runserver
